# Endpoint Backup Module SSE-KMS Example
Provides a SSE-KMS usage example for the endpoint backup module.

## Table of Contents

  - [Requirements](#requirements)

## Overview
Example module call uses S3 SSE-KMS server side encryption with a customer managed KMS key. State file should be encrypted and stored in a secure backend when enabling Access key creation without PGP encryption.
